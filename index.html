leaders.json 생성 / 복사
[GitHub에 붙여 넣는 방법]
1) GitHub에서 so.t-leader-choice 레포로 이동.
2) leaders.json 파일 클릭 → 우측 상단 연필(✏️ Edit) 버튼 클릭.
3) 파일 내용 전체 선택(Ctrl+A) 후 삭제 → 위 JSON 영역의 내용을 통째로 복사(Ctrl+C)하여 붙여 넣기(Ctrl+V).
4) 아래쪽 Commit changes 버튼을 눌러 저장.
5) 공개 페이지(https://wordycow.github.io/so.t-leader-choice/)에서 새로고침하면 모든 사용자가 동일한 리더 정보를 보게 됩니다.
